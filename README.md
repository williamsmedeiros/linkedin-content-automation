
# 🚀 LinkedIn Content Automation

Projeto de automação desenvolvido para gerar conteúdos para o LinkedIn utilizando Inteligência Artificial.

## 📖 Sobre o projeto

Este projeto foi desenvolvido durante meus estudos em Automação, Inteligência Artificial e Análise de Dados.

O objetivo é automatizar a geração de conteúdos a partir de assuntos cadastrados em uma planilha do Google Sheets.

O workflow identifica os assuntos pendentes, realiza uma pesquisa utilizando a API do Tavily, envia o contexto para o Google Gemini e, por fim, atualiza automaticamente a planilha com o conteúdo gerado e o status da tarefa.

## 🛠️ Tecnologias utilizadas

- n8n
- Google Sheets
- Tavily API
- Google Gemini
- HTTP Request
